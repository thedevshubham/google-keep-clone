# google-keep-clone
Google Keep Notes

### A client and server model to feed real-time notes.
#### Using graphql subscriptions, pub sub and web sockets.

* Client: React Js
* Server: Graphql

Steps to start on local env:
* Clone
* Run **npm install** on client and server.
* Run **npm start** on client and server.
